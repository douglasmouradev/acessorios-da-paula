# Acessorios da Paula

Site institucional e vitrine digital da **Acessorios da Paula**, com foco em moda feminina, seções informativas, navegação moderna e visual profissional.

## Visao geral

Este projeto foi desenvolvido como uma landing page completa para apresentar:

- colecoes e categorias de produtos;
- destaques e depoimentos;
- lookbook e area de newsletter;
- canais oficiais de atendimento (WhatsApp e Instagram).

## Tecnologias utilizadas

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`

## Estrutura do projeto

```text
.
├── index.html
├── style.css
└── script.js
```

## Como executar localmente

1. Clone o repositorio:

```bash
git clone https://github.com/douglasmouradev/acessorios-da-paula.git
```

2. Entre na pasta do projeto:

```bash
cd acessorios-da-paula
```

3. Abra o arquivo `index.html` no navegador.

> Dica: se usar VS Code/Cursor, pode abrir com extensao Live Server para recarregamento automatico.
